香水市场和社交媒体

彭博社这篇 2018 年的文章提到一个观点，Instagram 等图片社区的出现对彩妆和护肤品有积极影响，但与之相对的，人们无法隔着屏幕闻到香水的味道，这被用来解释为什么香水市场表现不如其他化妆品。

尽管从逻辑上看这很合乎情理，但仍然造成些困惑：难道我们能认为美颜相机的普及会损害化妆品的销量吗？如果延伸上述的话题，这可以抽象成一个新问题：一个新兴事物会如何影响原本世界？

1898 年，作家 Arthur 预测电话的普及能够带来哪些影响，这篇一百年前的文章非常有趣，阅读时会产生一种对历史的惊讶感：

  • 电话将歌剧这种富人社会的奢侈品大众化，人们在家通过电话就能听到另一头专业的歌剧演出；
  • 每个人都有一张电话时刻表，上面列出播放相应新闻专题的时间，新闻会在指定时间打来；
  • 报纸业最初将电话作为自己的替代品看，但很快发现电话会刺激人们购买报纸的欲望，因为人们想要了解更多新闻细节；
  • 有的人担心，人们会一整天抱着电话，担心错过任何一条新闻；
  • 最后，作者提到如果电话有一天能够传递视觉信息（即现如今的视频），那么人们会丧失旅行的源动力，因为当远处的一切都是已知的事实时，人们不会再有好奇心去探索。

作者提到电话促进新闻业的发展，却认为视频对旅行业会有消极影响，理由非常有趣。

另一个例子是盗版音乐。在 Napster 最火热的时候，音乐行业普遍认为 Napster 损害自己的盈利空间。当时有两位经济学家发表论文 The Effect of File Sharing on Record Sales: An Empirical Analysis，他们在对比大量下载样本和专辑的销售情况后，认为非法下载对音乐销售几乎没有任何影响：那些非法下载音乐的人并不是会为 CD 支付 15 美元的人。

与之相似观点是，在 Napster 中，用户通常会先浏览其他文件，并在文件服务器聊天室中讨论音乐，这种浏览的过程会促进新的 CD 销售。2000 年 7月，英国摇滚乐队 Radiohead 的专辑《Kid A》中的歌曲在专辑发布前三个月便流通进入了 Napster，随后这张专辑在 Billboard 200（一个显示美国最受欢迎的 200 张音乐专辑的排行榜）上占据榜首，而此前 Radiohead 在美国从未进入过前 20 名，这张专辑的成功似乎说明，Napster 是一个很好的音乐推广工具。


美团袋鼠耳朵背后的设计思考

起初我没有深入思考这次品牌营销，直到我在未来预想图的采访文章中看到这样一段内容：

其实，早在 2016 年，美团外卖就设计了袋鼠耳朵头盔，其灵感来源于美团外卖的 logo——一只奔跑的黄色袋鼠。美团外卖配送营销中心项目负责人铁镭告诉未来预想图，这个想法缘起于 2016 年，美团发现，“用户和骑手接触中没有眼神交流，用户关注餐有没有送错，骑手关心用户接好了没别洒了。”
“需要一个创意触发眼神的互动，还需要一个亮点能推动二度传播和内容共创。”铁镭补充说。这个创意经过 4 年优化，最终形成如今的“袋鼠耳朵”。

至少我很喜欢这个理由，这让我想到 EICO 的 Albert Cheng 在为丁香诊所做空间设计时，发现医生和到访用户初次见面常常陷入无话可聊的尴尬情景，于是想到通过握手解决破冰问题。但在中国文化中，握手并不是自然发生的礼仪动作，Cheng 想到可以将候诊室的座椅换成无印良品懒人沙发，这样的椅子既舒适，同时又让人近乎躺下，这样当第一次见面时，医生就能很自然地伸手拉起用户，顺势完成握手动作。

尽管随着数字产品的普及，许多人将用户体验设计等同于界面设计，但回忆下这个词的创造者唐 · 诺曼（Donald A. Norman）最初的想法，他认为人机界面和可用性的定义太狭隘了，应该拥有一个词语能够涵盖人们体验的所有方面，无论是工业设计图形，界面，物理交互还是使用手册。用户体验应该包括人们体验产品的方方面面，而不仅仅是数字领域的专有名词。

还可以继续追溯，早在公元前 5 世纪，希腊人就提出人类工程学（ergonomics），试图建立一套原则使人类工作更方便和高效。一个例子是西方医学之父希波克拉底（hippocrates），他详细描述了应该如何建造外科医生的工作场所：灯光、摆放工具的位置或是医生的姿势都是应该被考虑的因素，这和现如今用户体验设计的概念已经有非常类似的内核。

为什么 Apple 和 Google 是竞争关系？

这是一篇 2015 年文章，作者试图分析美国 7 家科技公司的敌友关系（Amazon、Apple、Facebook、Google、Microsoft、Twitter 和 Uber）。我对作者列举的案例存有一些疑问，但作者分类科技公司的方式值得记录一笔：

第一种分类方式：Information About X & Premium Enablers

  • Information About X：a set of information to be organized and understood.
  • Premium Enablers：a set of objectives and tasks to be accomplished.

第二种分类方式：Identity & Interface With the World

  • Identity：The user’s perspective on the world
  • Interface With the World：The world’s utility for the user

第三种分类法方式：Productivity Platforms & Organize the World’s X

  • Productivity Platforms： the principle problem to be solved is one of scarcity
  • Organize the World’s X：the principle problem to be solved is one of abundance

作者试图通过这种分类方法解释这些问题：

  • 为什么 Google 无法做出和微软一样优秀的办公产品？
  • 为什么微软的 Bing 无法像 Google 一样成功？
  • 为什么亚马逊做手机失败了？

创造的源动力

在 The Bus Ticket Theory Of Genius 中，作者认为天才的秘诀是对重要的事情拥有无私的痴迷。恰是因为这种痴迷，让人能够忘却过程的痛苦，恰是因为不带有目的性的，让人不关注短期收益。那什么是重要的事情？作者提到创造事物显然比消费一个存在的事物更重要。

但这就是我想说的，在我的观点里，创造事物的源动力不仅仅是痴迷一个事物，还需要另一个重要的因素——对现状的不满。一个痴迷于音乐的人可能听过上万首歌曲，但一个对现状不满的人才会思考为什么下载和管理音乐这么复杂，并想办法解决该问题（我指的是乔布斯和 iTunes）。

对现状的极度不满，并永远相信自己能够做的更好，我认为这才是创造的源动力。
